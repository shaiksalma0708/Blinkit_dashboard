# Blinkit Sales Dashboard (Excel) 📊

This project presents an interactive sales dashboard for Blinkit, created using Microsoft Excel. The dashboard provides insights into sales performance, customer behavior, and product trends.

## 📌 Objective
To analyze Blinkit's sales data and generate meaningful insights to support business decision-making and performance tracking.

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning & Transformation
- Conditional Formatting
- Slicers & Filters

## 📊 Key Features
- Interactive dashboard with slicers for dynamic filtering
- Sales analysis based on:
  - Product categories
  - Outlet size and location
  - Item type and fat content
- KPIs displayed:
  - Total Sales
  - Average Sales
  - Number of Items Sold
  - Average Rating
- Trend analysis of sales across different outlets

## 📈 Key Insights
- Certain product categories contribute more to overall sales
- Medium and large outlets generate higher revenue
- Specific item types show consistent customer demand
- Customer ratings influence product performance

## 📁 Project Files
- `Blinkit_Dashboard.xlsx` – Excel dashboard file
- `dataset.xlsx / csv` – Source dataset
- `README.md` – Project documentation

## 🚀 Conclusion
The dashboard helps in understanding sales patterns, identifying top-performing products, and improving business strategies using data-driven insights.
